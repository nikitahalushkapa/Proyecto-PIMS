## Descripción del Proyecto

En este trabajo nos centramos en el diseño e implementación de un sistema de copias de seguridad automatizado para que sea usado en multiplataformas, trabajando tanto con Windows y Ubuntu. Haciendo esto, nos permite garantizar que los datos se guarden de forma segura y eficiente.
